✦ Why FileDrop

Not just another
file transfer tool.

FileDrop was built from the ground up to solve a simple problem: transferring files between your devices without trusting a third party with your data.

End-to-End Encrypted

Every transfer is protected with ECDH key exchange and AES-256-GCM encryption. Even on shared Wi-Fi, your files are unreadable to anyone else.

⚡

LAN-Speed Transfers

Files travel directly between devices over your local network — not through a distant server. Expect speeds up to 300 MB/s on a good connection.

Zero Cloud, Zero Trace

Nothing is ever uploaded. No accounts, no sign-ups, no telemetry. Your files stay between your devices, period.

Cross-Platform

Works on Windows, macOS, and Linux. Send files from any laptop to any phone's browser — no app install needed on your phone.

Beautiful TUI

A polished, full-color terminal UI with live progress bars, QR codes, real-time file listings, and an interactive main menu — right in your terminal.

One-Click Install

Download the executable, run it, and it auto-installs itself to your system PATH. No package managers, no dependencies, no configuration files.

📡

Works Anywhere — Offline Mode

No Wi-Fi router or internet needed. Use your phone as a hotspot to transfer files in forests, flights, or anywhere offline. True peer-to-peer, zero infrastructure.

⌘ Setup Guide

Up and running in under 60 seconds

Follow these quick steps to install FileDrop and start transferring files.

1

Download the executable

Download filedrop-windows-amd64.exe from the download section above, or from the GitHub Releases page.

2

Run the executable

Double-click the downloaded .exe file. A beautiful interactive menu will appear. Select "Install FileDrop (Add to System PATH)" and press Enter.

If Windows SmartScreen shows a warning, click "More info" → "Run anyway". FileDrop is open-source and safe.

3

Open a new terminal & use FileDrop

After installation, open a new Command Prompt or PowerShell. You can now run FileDrop from anywhere:

filedrop receive
4

Transfer files from your phone

Open the QR code URL on your phone's browser (same Wi-Fi network). Upload files directly to your laptop — or download files from it.

1

$ Terminal

All commands at a glance

FileDrop is fully controlled from the terminal. Here's everything you can do.

$ filedrop receive
Receive files from your phone into the current directory. Opens a TUI with QR code, progress bars, and live file listing.
$ filedrop share
Share files from the current folder. Generates a secure download link accessible from your phone's browser.
$ filedrop pair
Pair a new device using mDNS discovery and TLS certificate exchange. One-time setup for trusted connections.
$ filedrop peers
List all previously paired devices with their name, IP, and last-seen timestamp.
$ filedrop hotspot
Create a direct Wi-Fi hotspot for peer-to-peer transfers when no router is available.
$ filedrop --help
Display all available commands, flags, and usage examples.
