## ✨ Features

- **Premium Dark Theme** — Glassmorphism cards, 3D transforms, deep navy-to-black palette
- **3D Animated Hero** — Spinning CSS cube with orbiting rings, glowing dots, and floating labels — zero images
- **Auto OS Detection** — Highlights the correct download card for the visitor's platform
- **Tabbed Installation Guide** — OS-specific step-by-step instructions with copy-to-clipboard code blocks
- **Comparison Table** — FileDrop vs Google Drive, SHAREit, Nearby Share, AirDrop, Bluetooth, and USB
- **Animated Feature Icons** — Pure CSS 3D icons with pulse, glow, and blink animations
- **Responsive Design** — Optimized for mobile, tablet, and desktop
- **Scroll-Reveal Animations** — Staggered entry animations using Intersection Observer
- **Particle Background** — Canvas-based network with dynamic node connections
- **Zero Dependencies** — Pure HTML, CSS, and vanilla JavaScript — no frameworks, no build step
- **Offline Mode** — Works without a Wi-Fi router or internet — use your phone as a hotspot in forests, flights, or anywhere

## 🎨 Design Highlights

| Element | Implementation |
|---------|---------------|
| **Background** | Canvas-based particle network with dynamic connections |
| **Hero** | 3D CSS cube + orbiting rings/dots + floating labels (no images) |
| **Cards** | Glassmorphism (`backdrop-filter: blur`) with gradient borders on hover |
| **Icons** | Pure CSS animated shapes (shield, bolt, eye, terminal, devices) |
| **Typography** | Inter (sans-serif) + JetBrains Mono (code) from Google Fonts |
| **Colors** | Deep navy `#050a14` · Green `#00e87b` · Teal `#00d4ff` · Purple `#8b5cf6` |
| **Animations** | CSS keyframes + Intersection Observer scroll-reveal + staggered delays |
| **3D Effects** | `perspective` + `rotateX/Y` + `transform-style: preserve-3d` on cube and cards |
